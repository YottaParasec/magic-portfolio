## Overview

Designed and built a SaaS MVP for psychometric and cognitive assessments in hiring, featuring a custom question and logic builder, analytics dashboards, and a candidate portal. The platform streamlines candidate evaluation and improves hiring outcomes.

## Key Features

*   Custom question and logic builder for psychometric and cognitive assessments.
*   Analytics dashboards for visualizing candidate performance.
*   Candidate portal for a seamless assessment experience.
*   Streamlines candidate evaluation and improves hiring outcomes.

## Challenges and Learnings

Conceived, validated, and designed an MVP for a SaaS platform enabling data-driven psychometric candidate screening. Led stakeholder interviews with HR and organizational psychology professionals, steered market surveys and pilot outreach. Documented requirements, mapped recruiter workflows, and developed user scenarios for dashboard design. Built early user engagement for launch.

## Outcome

The platform aims to streamline candidate evaluation and improve hiring outcomes by providing a comprehensive tool for psychometric and cognitive assessments.
